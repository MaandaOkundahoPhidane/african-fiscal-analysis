A comprehensive data analysis project examining budget deficits and surpluses across 14 African countries from 1960 to 2025. This project transforms raw fiscal data into actionable insights for policymakers, economists, and researchers.

## üìä Project Overview

This analysis provides deep insights into fiscal health patterns, trends, and country performance across Africa. The project processes macroeconomic data to identify chronic deficit issues, volatility patterns, and strategic recommendations for fiscal policy improvement.

### Key Features

- **Multi-country Analysis**: 14 African nations spanning 65+ years
- **Trend Identification**: Year-over-year fiscal performance tracking
- **Risk Assessment**: Country categorization by deficit severity and volatility
- **Strategic Recommendations**: Data-driven policy suggestions
- **Professional Visualizations**: Comprehensive dashboard and charts
- **Exportable Reports**: Excel and image outputs for presentations
