1. PROJECT OVERVIEW
===================

Purpose:
--------
This project analyzes budget deficit/surplus patterns across 14 African countries
from 1960 to 2025. The analysis provides insights into fiscal health, identifies
trends, and offers data-driven policy recommendations.

Scope:
------
- 14 African countries
- Time period: 1960-2025
- 23,782 data records
- Mixed frequency data (Yearly, Monthly, Quarterly)

Countries Analyzed:
-------------------

2. Angola

4. Egypt
5. Ethiopia
6. Ghana
7. Ivory Coast
8. Kenya

11. Senegal
12. South Africa
13. Tanzania

3. ANALYSIS METHODOLOGY
=======================

Key Metrics Calculated:
-----------------------
1. Deficit Frequency: Percentage of periods with negative balances
2. Average Balance: Mean fiscal position per country
3. Volatility: Standard deviation of budget balances
4. Trend Analysis: Year-over-year changes using linear regression
5. Risk Categorization: High/Medium/Low based on deficit patterns

Statistical Methods:
--------------------
- Linear regression for trend analysis
- Time-series interpolation for missing data
- Standard deviation for volatility measurement
- Correlation analysis for country comparisons
- Kernel Density Estimation (KDE) for distribution analysis

Country Categorization:
-----------------------
Chronic Deficit: >75% deficit frequency
Large Deficit: Average deficit > $50 million
High Volatility: Std dev > median * 2
Improving Trend: Positive slope with R² > 0.3
Deteriorating Trend: Negative slope with R² > 0.3

6. INTERPRETATION GUIDE
=======================

Understanding the Metrics:
--------------------------

Deficit Frequency:
- 0-30%: Healthy fiscal management
- 30-60%: Moderate fiscal challenges
- 60-75%: Significant fiscal issues
- 75-100%: Chronic deficit problems

Average Balance:
- Positive: Budget surplus (healthy)
- Negative: Budget deficit (concerning)
- Near zero: Balanced budget (ideal)

Volatility:
- Low: Predictable fiscal environment
- High: Unstable fiscal management
- Very High: Economic uncertainty

Trend Analysis:
- Improving: Positive slope, fiscal health getting better
- Deteriorating: Negative slope, fiscal health worsening
- Stable: Minimal change over time

Risk Levels:
------------
Low Risk: Deficit frequency < 40%, stable trends
Medium Risk: Deficit frequency 40-70%, some volatility
High Risk: Deficit frequency > 70%, high volatility

8. TROUBLESHOOTING
===================

Common Issues and Solutions:

1. "TypeError: '<' not supported between instances of 'str' and 'int'"
   Solution: Ensure Amount column is properly converted to numeric in cleaned data

2. "FileNotFoundError: cleaned_budget_data.csv"
   Solution: Run data cleaning script first or ensure cleaned files are in correct directory

3. Memory errors with large datasets
   Solution: Process data in chunks or increase system RAM

4. Visualization display issues
   Solution: Ensure matplotlib backend is properly configured

5. Excel export failures
   Solution: Install openpyxl: pip install openpyxl

Data Validation:
----------------
- Check that all Amount values are numeric
- Verify date ranges are reasonable
- Ensure country names are consistent
- Confirm no duplicate records exist
